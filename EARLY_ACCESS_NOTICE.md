# Early Access Notice

BayouOps Suite Pro Developer Preview v0.2 is an early access package for review,
testing, and feedback.

This release includes lightweight Windows and Linux collectors that generate
local operational readiness and health exports. The collectors are intended to
help small IT teams and administrators evaluate the reporting workflow before a
broader release.

## Preview Scope

- Developer Preview only
- Local-only collection
- Read-only commands
- No destructive actions
- No managed service or enterprise support commitment

## Expectations

Some output may vary by operating system version, local permissions, installed
tools, and host configuration. Missing commands or inaccessible data should be
reported in the generated output where possible.

Review scripts before running them in environments with strict change control.
