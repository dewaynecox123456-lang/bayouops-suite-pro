# BayouFinds Aqua Pro v1.0 Fedora 42 GNOME Desktop Standard

BayouFinds Aqua Pro v1.0 is a package-only Fedora 42 GNOME desktop standard for BayouFinds/BayouOps workstations.

It stages a complete branded desktop system for review and controlled installation planning without applying changes automatically.

## Included

- GNOME icon theme staging: `assets/icons/BayouFinds-Aqua-Pro/`
- Cypress Gray default folders
- Bayou Green BayouFinds, BayouOps, and Projects folders
- River Blue Infrastructure folders
- Copper Archives folders
- Wallpaper and lock screen assets
- GDM login assets staged for review only
- Plymouth boot assets staged for review only
- Browser homepage documentation
- Dock layout documentation
- Home directory folder mockup
- Product screenshots
- Validation script
- Release notes
- BayouFinds Aqua Pro Standards Guide

## Safety

This v1.0 package does not install anything by itself.

During packaging and validation:

- No sudo commands are required.
- No dconf writes are performed.
- No GDM configuration is modified.
- No Plymouth configuration is modified.
- No system files are changed.
