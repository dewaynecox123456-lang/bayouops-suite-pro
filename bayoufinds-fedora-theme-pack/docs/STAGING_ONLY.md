# Staging Only

BayouFinds Aqua Pro v1.0 includes assets that are staged for future controlled installation.

## Staged Assets

- GDM login background: `login/gdm-background.png`
- Plymouth boot theme: `plymouth/bayoufinds/`
- Firefox homepage policy reference: `assets/browser/bayoufinds-homepage.json`

## Not Applied By This Package

This package does not:

- Run sudo
- Write dconf
- Modify GDM
- Modify Plymouth
- Copy files into `/usr/share`
- Change browser policy paths

System application requires a separate approved change process.
