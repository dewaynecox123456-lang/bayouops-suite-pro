# File Type Icons

BayouFinds Aqua Pro v1.1 adds user-level file-type icon assets for common operations and development workflows.

## Covered Types

- PowerShell: `.ps1`, `.psm1`, `.psd1`
- Shell: `.sh`, `.bash`, `.zsh`
- JSON: `.json`
- YAML: `.yaml`, `.yml`
- Markdown: `.md`
- PDF: `.pdf`
- Archives: `.zip`, `.tar`, `.tar.gz`, `.7z`
- Images: `.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`
- Web: `.html`, `.css`, `.js`
- Node/package: `package.json`
- Git/repo folders
- BayouOps reports, exports, and snapshots

## Icon Theme Paths

Scalable SVG assets:

```text
assets/icons/BayouFinds-Aqua-Pro/mimetypes/scalable/
```

PNG exports:

```text
assets/icons/BayouFinds-Aqua-Pro/mimetypes/256/
assets/icons/BayouFinds-Aqua-Pro/mimetypes/512/
assets/icons/BayouFinds-Aqua-Pro/mimetypes/1024/
```

## Notes

GNOME file managers use MIME type icon names rather than raw filename extensions. This package includes common MIME aliases where practical, but exact extension-to-icon behavior depends on the local MIME database and file manager.
