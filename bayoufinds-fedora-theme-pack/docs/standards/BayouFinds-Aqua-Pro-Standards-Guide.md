# BayouFinds Aqua Pro Standards Guide

## Purpose

BayouFinds Aqua Pro v1.0 provides a professional Fedora 42 GNOME desktop identity for BayouFinds/BayouOps workstations.

## Visual Standard

- Dark graphite desktop foundation
- Louisiana bayou atmosphere
- Swamp green accents
- Steel/River Blue operational highlights
- Polished aqua folder styling
- Clean enterprise appearance
- No gaming or RGB styling

## Folder Color Standard

- Default folders: Cypress Gray
- BayouFinds folders: Bayou Green
- BayouOps folders: Bayou Green with subtle emblem accent
- Project folders: Bayou Green
- Infrastructure folders: River Blue
- Archive folders: Copper

## Desktop Standard

- Wallpaper: `wallpapers/bayoufinds-wallpaper-4k.png`
- Lock screen: `lockscreen/bayoufinds-lockscreen.png`
- Icon theme: `assets/icons/BayouFinds-Aqua-Pro/`
- Login screen asset: staged only
- Plymouth asset: staged only

## Home Directory Standard

Recommended home folder examples:

- `BayouFinds`
- `BayouOps`
- `Projects`
- `Infrastructure`
- `Archives`
- General folders use the Cypress Gray default folder.

See `assets/generated/v1.0/home-mockup/home-directory-standard.png`.

## Browser Standard

Recommended homepage:

```text
https://bayoufinds.com
```

See `docs/BROWSER_HOMEPAGE.md`.

## Dock Standard

Keep the dock restrained and operations-focused. See `docs/DOCK_LAYOUT.md`.

## Safety Standard

No autonomous workstation changes are part of this standard. Operators review, approve, and apply all workstation changes.
