# Dock Layout

BayouFinds Aqua Pro v1.0 recommends a restrained GNOME dock layout for operations workstations.

## Recommended Favorites

- Files
- Terminal
- Firefox
- Text Editor
- Settings
- System Monitor

## Optional Operations Favorites

- Logs or document viewer
- Password manager approved by the organization
- Remote-console viewer approved by the organization

## Safety

This package does not write GNOME dock settings. Operators should apply dock layout changes manually or through an approved device-management process.
