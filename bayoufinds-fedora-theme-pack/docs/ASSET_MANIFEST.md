## Generated Base Image Prompt

Use case: ads-marketing

Asset type: Fedora 42 enterprise workstation background base image, 16:9, suitable for 4K wallpaper and login lock screens.

Primary request: Create a professional BayouFinds/BayouOps-style enterprise operations center visual background with no text and no logos.

Scene/backdrop: dark graphite Louisiana bayou atmosphere blended with subtle infrastructure graphics, faint server racks, monitoring dashboard silhouettes, and low-contrast network topology lines.

Subject: clean Fortune 500 operations workstation mood, swamp green accents and steel blue highlights.

Composition: leave clean negative space in the upper-left and center-left for deterministic logo and text overlays later; balanced depth, restrained contrast.

Style: enterprise grade, polished, realistic but subtle, professional operations center feel.

Avoid: no words, no letters, no logos, no gaming theme, no hacker theme, no neon cyberpunk, no people, no skulls, no mascots, no watermarks.
