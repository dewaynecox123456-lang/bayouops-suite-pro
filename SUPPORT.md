# Support

BayouOps Suite Pro Developer Preview v0.2 includes basic early access support
guidance.

## Before Requesting Help

Please include:

- Operating system and version
- PowerShell or Bash version
- Command used to run the collector
- Error message or generated report excerpt
- Whether the script was run as a standard user or administrator/root

## Scope

Support for this Developer Preview is limited to:

- Usage questions
- Reproducible script errors
- Documentation corrections
- Feedback on report fields and export format

This Developer Preview does not include enterprise support, guaranteed response
times, remote administration, production monitoring, or service-level
commitments.

## Safety

The included collectors are designed to be local-only and read-only. Review the
scripts before use and run them only on systems you own or are authorized to
administer.
