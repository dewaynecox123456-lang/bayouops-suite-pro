Software / Agent Visibility

Operational awareness for endpoint software drift.

Review agent coverage, stale check-ins, missing installs, version drift, and export-ready evidence for leadership, CAB, audit, and handoff discussions. Future real inventory data can come from the read-only Windows Software Inventory Collector for BayouOps reporting.

Patch / KB Visibility

Operational readiness for Windows patch evidence.

Review missing KBs, pending reboot signals, stale patch timelines, and OS support drift from read-only patch inventory data. This module is visibility/reporting only; it does not deploy patches, trigger Windows Update, reboot systems, remediate endpoints, or perform remote execution.
