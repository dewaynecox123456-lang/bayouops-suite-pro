# Known Limitations

BayouOps Suite Pro is currently in controlled Dev Tester / early access review.

## Current Limitations

- Not intended for production-critical decisions without independent validation.
- No online activation, customer portal, telemetry, or automatic update system.
- License files are generated manually and delivered offline.
- PDF generation is not implemented.
- ZIP packaging is prepared manually after package approval.
- Payhip delivery is documentation-only prep at this stage.
- Some workflows require Node.js for local demo/report pack generation.
- Aggregation requires Python 3.
- Windows inventory fields may be limited by local policy or permissions.
- Linux health exports depend on local command availability.

## Safety Boundaries

BayouOps does not:

- modify endpoints
- deploy agents
- trigger remediation
- run background services
- collect telemetry
- phone home
- start hidden polling loops
