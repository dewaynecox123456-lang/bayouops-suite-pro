# BayouOps Color Palette

| Token | Hex | Use |
| --- | --- | --- |
| Midnight Ops | `#0B1220` | Dark UI backgrounds, launcher surfaces, social banners |
| Command Navy | `#142033` | Secondary dark panels and documentation headers |
| Field Steel | `#334155` | Borders, dividers, secondary UI |
| Signal Teal | `#1FB6A6` | Primary accent, active telemetry, calls to action |
| Readiness Cyan | `#38BDF8` | Radar arcs, highlights, links |
| Bayou Copper | `#C47A3A` | Subtle Southern heritage, navigation path, restrained warmth |
| Live Oak Moss | `#4D7C59` | Secondary heritage accent, status context |
| Delta Mist | `#E5EDF5` | Light text on dark backgrounds |
| Whitewater | `#F8FAFC` | High-contrast text and icon detail |

Use teal and cyan for operational clarity. Use copper sparingly so the Bayou identity is present but quiet.
