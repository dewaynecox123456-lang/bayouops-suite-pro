# Demo Assets Checklist

## Dashboard Screenshots

- [ ] Main BayouOps dashboard or operational summary view.
- [ ] Readiness overview with red, yellow, and green states visible.
- [ ] High-risk or critical-risk example.
- [ ] Stale-system summary.
- [ ] Unsupported-system example.
- [ ] SSL or certificate findings view.
- [ ] Executive summary or management-ready status view.
- [ ] Export directory or generated report list.

## Exports Needed

- [ ] Patch readiness CSV.
- [ ] Operational risk summary text file.
- [ ] Executive operational summary CSV.
- [ ] Linux health summary CSV, if included in the demo.
- [ ] Windows readiness export example, if included in the demo.
- [ ] Before-window export set.
- [ ] After-window or follow-up export set, if available.

## PDF and Report Examples

- [ ] PDF-ready executive summary.
- [ ] CAB review attachment example.
- [ ] SOX or audit evidence example.
- [ ] Exception review summary.
- [ ] Stale-system validation list.
- [ ] Certificate findings summary.

## Dashboard States

- [ ] Mostly healthy environment with a few review items.
- [ ] Mixed readiness state showing red, yellow, and green examples.
- [ ] Critical-risk state with a clear reason.
- [ ] Stale-system-heavy view.
- [ ] Certificate-risk-focused view.
- [ ] Management summary view with plain-language status.

## Red, Yellow, and Green Examples

- [ ] Green: system current, evidence recent, no immediate action required.
- [ ] Yellow: needs review due to stale evidence, pending owner validation, or moderate finding.
- [ ] Red: critical issue, unsupported system, expired certificate, or major readiness blocker.

## Critical-Risk Examples

- [ ] Unsupported operating system.
- [ ] Server not reporting recently.
- [ ] Expired SSL certificate.
- [ ] High-priority system missing readiness evidence.
- [ ] Exception without owner or review date.

## Stale-System Examples

- [ ] Server with old last-seen date.
- [ ] Server listed in inventory but missing recent patch evidence.
- [ ] Retired or unknown system still present in reporting.
- [ ] Business-critical server requiring owner confirmation.

## SSL Findings

- [ ] Expired certificate.
- [ ] Certificate expiring soon.
- [ ] Unknown or incomplete certificate metadata.
- [ ] Customer-facing service certificate finding.
- [ ] Internal service certificate finding.

## Before and After Visuals

- [ ] Before: scattered spreadsheets or multiple raw export files.
- [ ] Before: unclear readiness status.
- [ ] After: consolidated dashboard.
- [ ] After: readiness score or status categories.
- [ ] After: executive summary export.
- [ ] After: CAB-ready evidence package.

## Branding and Presentation

- [ ] BayouOps logo or lockup.
- [ ] Clean terminal or launcher view.
- [ ] Consistent file naming in exports.
- [ ] No sensitive hostnames, IP addresses, user names, or customer data.
- [ ] Demo data labeled clearly as sample or sanitized where appropriate.
