# Demo Video Capture Inserts

These PNG files are polished operational inserts for the BayouOps Suite Pro demo video. They are designed as clean, dark-theme dashboard visuals that support the "Operational Readiness & Visibility" positioning.

## Files

- `01-readiness-score.png` - Use when introducing the overall readiness view. It shows patch readiness, deployment visibility, endpoint coverage, risk level, and a readiness trend.
- `02-ssl-findings.png` - Use during the SSL and certificate visibility section. It highlights expiring certificates, expired certificates, unsupported TLS, affected systems, and severity-based review.
- `03-stale-systems.png` - Use when explaining operational drift. It shows stale endpoints, unsupported operating systems, old agent versions, last check-in status, and affected systems.
- `04-executive-export.png` - Use during the reporting workflow section. It shows how operational evidence becomes PDF, CSV, and audit/CAB-ready summary output for management review.

## Usage Notes

Use these as static inserts, zoom targets, or transition slides between live dashboard footage. They are intentionally written with realistic operational labels and avoid replacement claims, hype language, and fake "AI-powered" positioning.
