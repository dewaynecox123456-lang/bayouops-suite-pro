BAYOUOPS QUICK START
=================================

1. Extract the ZIP file first.
2. Open the BayouOps folder.
3. Double-click BayouOps-Launcher.bat on Windows, or open the docs for a manual workflow.
4. Choose an operator-triggered action from the launcher menu:
   - Generate Demo Environment
   - Render Executive Dashboard
   - Generate Executive Export Pack
   - Open Documentation
   - Open About BayouOps
   - Open Support Information
   - Open Terms / License Agreement
5. Review generated exports and reports locally.

Exports are saved under:
./exports/

Network Inventory Collector:
BayouOps includes a Windows network inventory collector for approved target
lists. It uses a CSV file instead of blind network scanning.

Collector:
./collectors/windows/Invoke-BayouOpsNetworkInventory.ps1

Sample target CSV:
./samples/windows-network-targets.sample.csv

Documentation:
./docs/WINDOWS_NETWORK_INVENTORY_COLLECTOR.md

This collector is read-only and intended only for systems you own or are authorized to manage.

Network readiness reporting workflow:
Run collector → generate network readiness dashboard → review executive summary.

1. Run the collector from an authorized Windows admin workstation.
2. Place the collector output at:
   ./exports/bayouops-network-inventory.csv
3. Run the reporting pipeline:
   node scripts/build-network-readiness.mjs
4. Review:
   ./exports/network-readiness-dashboard.html
   ./exports/network-readiness-summary.md
   ./exports/network-readiness.csv

If collector output is missing, BayouOps uses sample demo data so the dashboard
and executive summary workflow can still be reviewed safely.

Executive demo packs, when generated from the source repo, are saved under:
./exports/demo/

Lines of Business can be edited locally in:
./config/lines-of-business.json

A license file is required for protected workflows.
Place customer license files at:
./config/license.json

Contact support@bayoufinds.com if you need a license file.
