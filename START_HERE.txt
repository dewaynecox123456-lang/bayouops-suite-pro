BAYOUOPS QUICK START
=================================

1. Extract the ZIP file first.
2. Open the BayouOps folder.
3. Double-click BayouOps-Launcher.bat on Windows, or open the docs for a manual workflow.
4. Choose an operator-triggered action from the launcher menu:
   - Generate Demo Environment
   - Render Executive Dashboard
   - Generate Executive Export Pack
   - Open Documentation
   - Open About BayouOps
   - Open Support Information
   - Open Terms / License Agreement
5. Review generated exports and reports locally.

Exports are saved under:
./exports/

Executive demo packs, when generated from the source repo, are saved under:
./exports/demo/

Lines of Business can be edited locally in:
./config/lines-of-business.json

A license file is required for protected workflows.
Place customer license files at:
./config/license.json

Contact support@bayoufinds.com if you need a license file.

conditions before protected workflows run. Review:
